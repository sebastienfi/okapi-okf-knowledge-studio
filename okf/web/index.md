# Web Feature

* [Graph canvas](graph-canvas.md) - The force-directed document graph: one node per file, colored by type, sized by degree, with hover-neighborhood highlighting.
* [Detail panel](detail-panel.md) - Rendered markdown with frontmatter meta and neighbor navigation; internal links move through the graph instead of loading pages.
* [Editor](editor.md) - In-place CodeMirror editing with edit/split/preview modes and conflict-safe saves.
* [Insights](insights.md) - Bundle health computed client-side: disconnected groups, orphans, broken links, conformance, stale timestamps.
* [Ask](ask-ai.md) - Opt-in AI answers grounded in the bundle, streamed with citations that light up the graph.
